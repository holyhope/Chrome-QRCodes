# Chrome-QRCodes

Extension for Google Chrome wich add a qrcode to pages.

# Install

1. Open Chrome browser.
2. Go to https://chrome.google.com/webstore/detail/qrcodes/mbbkfmdmpcjoocpfceeedkmoncmgnbim
3. Click on *add to chrome*
4. Enjoy !
